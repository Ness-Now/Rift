# T001 Foundation

## Objective

Establish the monorepo structure, runnable frontend and backend shells, initial worker scaffold, and the first documentation set.

## Context

The project starts from an empty repository and needs a clean foundation before any product logic is implemented.

## Scope

- Create repository structure for apps, packages, infra, docs, and tickets
- Bootstrap Next.js frontend shell with Tailwind
- Bootstrap FastAPI service shell with health and version routes
- Bootstrap worker package with placeholder job execution
- Write foundational product and architecture documents

## Non-Goals

- Riot API integration
- OpenAI integration
- Database modeling
- Authentication implementation
- Real dashboard features

## Definition Of Done

- The repository structure matches the agreed foundation layout
- Web and API apps can be installed and run locally
- Documentation explains intended system boundaries
- Follow-on tickets are present for the next iterations
